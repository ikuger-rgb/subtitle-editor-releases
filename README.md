# עורך כתוביות אוניברסלי

אפליקציית Windows לעריכת כתוביות עם תמלול אוטומטי בעברית.

## תכונות

- טעינת וידאו/אודיו (MP4, WebM, MKV, MP3, WAV ועוד)
- עריכת כתוביות עם ציר זמן אינטראקטיבי
- תמלול אוטומטי IVRIT AI (WebGPU — פועל מקומית על ה-GPU)
- עיצוב כתוביות: 3 פונטים עבריים, צבע, מתאר שחור, רקע
- ייצוא SRT / SBV / VTT
- ייצוא MP4 עם כתוביות שרופות
- תמיכה מלאה ב-RTL עברית
- עדכונים אוטומטיים

## הורדה

ראה את [Releases](../../releases) להורדת הגרסה האחרונה.

## בנייה מקוד מקור

```bash
npm install
npm run build
```

הדרישה: Node.js 18+
